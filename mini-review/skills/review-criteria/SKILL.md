# 코드 리뷰 기준

## 필수 체크 항목

### 1. 변경 영향도
- 이 변경이 다른 곳에 영향을 주는가?
- 한 곳만 수정하면 되는 구조인가?

### 2. 타입 안전성
- any 타입 사용 여부
- 판별 유니온 vs 플래그 조합

### 3. 가독성
- 함수 30줄 이하
- 파라미터 3개 이하
- 조기 반환 사용

### 4. 과한 추상화
- 패턴이 2-3번 반복 전에 추상화했는가?
- 불필요한 파일 분리가 있는가?

## 심각도

| 등급 | 기준 |
|------|------|
| CRITICAL | 버그, 보안 취약점 |
| HIGH | 원칙 위반, 유지보수 문제 |
| MEDIUM | 가독성, 네이밍 |
| LOW | 스타일, 선호도 |
