세션 종료 시 작업 컨텍스트를 `.ai/current.md`에 저장한다.

## 동작

### $ARGUMENTS에 "완료" 또는 "done" 포함 시
→ `.ai/current.md` 삭제 후 "작업 완료 처리됨" 출력.

### `.ai/current.md`가 없을 때
→ 대화 컨텍스트에서 아래 정보를 수집하여 새로 생성:

```markdown
# {작업 제목}

## 목표
{한 줄 요약}

## 스펙 참조
{.ai/specs/ 파일 경로, 없으면 이 섹션 생략}

## 진행
- [x] {완료된 것}
- [ ] {남은 것}

## 결정사항
- {결정 + 이유}

## 다음 할 일
{세션 재개 시 바로 시작할 것, 구체적으로}

## 메모
{블로커, 주의사항 등. 없으면 이 섹션 생략}
```

### `.ai/current.md`가 있을 때
→ 기존 내용을 읽고 아래 항목을 업데이트:
- 진행 체크리스트 (완료 항목 체크, 새 항목 추가)
- 결정사항 (새 결정 추가)
- 다음 할 일 (현재 시점 기준으로 갱신)
- 메모 (블로커/주의사항 갱신)

## 원칙
- 간결하게 (빠른 파악용)
- 코드 복붙 금지 (경로만 기록)
- 결정 이유 필수
- 항상 전체 덮어쓰기 (최신 상태 유지)

## 대화 출력
저장 완료 후 한 줄 요약만 출력:
```
.ai/current.md 저장 완료 — {작업 제목} ({완료/진행중})
```
